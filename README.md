# CTSCP
Control de teclado secundario con python en linux.

Este teclado secundario es totalmente independiente del primero o principal-
   cuando asignamos en este script una funcion, macro o acceso directo a cualquier tecla del teclado secundario.
   El mayor problema es la falta de manos para manejar los dos teclados al mismo tiempo, aun asi puede ser util.

Dependencia:
   evdev
   
Documentación dependencia:
   https://python-evdev.readthedocs.io/en/latest/
